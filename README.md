# PoolPay - Sistema de Gestión de Pagos

Sistema completo de gestión de clientes, facturas y pagos con integración de MercadoPago.

## 🚀 Inicio Rápido

### Requisitos
- Python 3.8+
- Node.js 16+
- Git

### Instalación y Ejecución

1. Clona el repositorio:
```bash
git clone https://github.com/valencastaldi/TPS-APP.git
cd TPS-APP
```

2. Ejecuta el proyecto completo:
```bash
run.bat
```

## 📁 Estructura del Proyecto

- `poolpay-backend/` - API REST con FastAPI
- `poolpay-frontend/` - Interfaz con React + TypeScript + Tailwind
- `run.bat` - Script para ejecutar backend y frontend simultáneamente

## 📚 Documentación

- Backend: Ver `poolpay-backend/README.md`
- Frontend: Ver `poolpay-frontend/README.md`
- API: `http://localhost:8000/docs` (al ejecutar)

## 🔧 Tecnologías

**Backend:**

**Frontend:**
